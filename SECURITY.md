# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Do **not** open a public GitHub issue for security-related problems.

Instead, contact the maintainer:

* Email: [vladislav.skidan@gmail.com](mailto:vladislav.skidan@gmail.com)

Please include:

* Description of the issue
* Steps to reproduce
* Potential impact

## Best Practices for Users

To ensure safe usage:

* Avoid modifying engine files without understanding the code
* Keep your Python environment secure
